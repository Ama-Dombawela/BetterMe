Home Page


Technical Description

The "BetterMe" website is dedicated to promoting good health and well-being. It features a navigation bar linking to key pages, including Home, Volunteer, Profile, Feedback, Sitemap, and Team. The homepage highlights the site's mission to educate individuals on mental health, physical health, and nutrition. Four content cards offer brief descriptions of these topics with links to more detailed information. The footer provides contact details, social media links, and additional navigation options. The website is designed with external stylesheets and JavaScript for a user-friendly and interactive experience.

Accessibility

The implemented web page incorporates accessibility by ensuring a clear structure, responsive design, and interactive elements that are keyboard-friendly. Buttons and interactive components have hover effects for better visibility, while images should include `alt` text to assist screen readers. The design also adapts to different screen sizes, improving readability across devices. To enhance accessibility further, adding focus indicators and an option to reduce animations would make the page more inclusive for users with disabilities.

# 🌿BetterMe

BetterMe is a wellness-focused website created to promote **Good Health and Well-Being** through practical content on mental health, physical health, nutrition, and healthy lifestyle habits. It was developed by three developers as a university project.

The project includes interactive pages such as a feedback form with preview/validation, a user profile page, team details, a volunteer page, and a health comparison table.

## 🎯Project Goals

- Share useful health and wellness information in a simple, visual format.
- Encourage healthy lifestyle choices through engaging content.
- Provide a clean multi-page website experience built with core web technologies.

## ✨Features

- Multi-page website navigation with a consistent header/footer.
- Splash screen entry page with auto-redirect to the home page.
- Home page with category cards:
	- **Mental Health** - Information on emotional and psychological well-being, stress management, meditation practices, and maintaining mental wellness
	- **Physical Health** - Exercise routines, fitness tips, body wellness, disease prevention, and maintaining physical strength
	- **Healthy Lifestyle** - Work-life balance guidance, daily healthy habits, avoiding harmful behaviors, and sustainable well-being practices
	- **Nutrition and Diet** - Healthy eating guides, balanced diet plans, nutritional information, portion control tips, and meal planning resources
- Feedback form with:
	- Built-in form validation
	- Character counter
	- Preview before confirm
- User profile page with goals, interests, and volunteering sections.
- Team page with contributor roles and implemented sections.
- Structured table page for quick health topic comparison.
